# 기능
- 회원 가입 및 로그인으로 웹 서비스 이용
- 마이페이지에서 내가 가진 재료를 냉장고에 추가 및 삭제 
- 마이페이지에서 내가 쓴 글 및 즐겨찾기 한 칵테일 확인 가능 
- 칵테일 사진 클릭 시 제조법 설명 및 내가 가지지 않은 재료에 대한 표시를 따로 해줌
- 게시판 기능을 통해 자유로운 정보 공유 가능
- 데일리 추천 칵테일 추천 시스템
- 키워드로 칵테일 레시피 검색 및 게시글 검색
